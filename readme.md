# Bug Triage and Prioritization using Machine Learning

This project automates bug classification, priority prediction, and duplicate detection using NLP and Logistic Regression.

## Features
- Classifies bug reports into categories (UI, Backend, Functional, Security, etc.)
- Predicts priority levels (High, Medium, Low)
- Detects duplicate bug reports using cosine similarity

## Tech Stack
- Python
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression
- Pandas, NumPy, Matplotlib
